# NLP Mastery Resource

### This is an Natural Language Processing Mastery Resource divided into 5 Phases, each phase covers conceptual blocks, necessary advance concepts used under classical NLP modeling, Deep Learning for NLP. Which will help you to decide when not to use LLMs, production optimization in modern day AI Systems. 

---

## Following are the things each Phase covers

### Phase 1 - Linguistic + Text Processing Core

- Text Normalization
- Tokenization theory
- N-grams
- Bag-of-Words
- TF-IDF
- Zipf's Law
- Lanugage Modeling basics

### Phase 2 - Classical NLP Modeling

- Naive Bayes
- Logistic Regressing for text
- SVM for text
- HMM
- CRF
- POS tagging
- NER (classical way)

### Phase 3 - Deep Learning NLP

- Word2Vec
- GloVe
- FastText
- RNN
- LSTM
- GRU
- Seq2Seq
- Attention

### Phase 4 - Transformers (Deep Dive)

- Self-attention math
- Multi-head attention
- Positional encoding
- Encoder vs Decoder
- BERT vs GPT objective
- Fine-tuning strategies

### Phase 5 - Modern Efficient NLP Engineering

- When not to use LLM
- Hybrid Pipelines
- Evaluation metrics (BLEU, ROGUE, F1, perplexity)
- Model compression
- Distillation
- Prompt vs fine-tune tradoffs
