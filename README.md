# NLP + GenAI Mastery

This repository is a structured, end-to-end learning system for mastering Natural Language Processing with a strong focus on modern Generative AI, Large Language Models, and AI Agent systems.

The goal of this repository is not just to study NLP concepts, but to develop the ability to design, build, and optimize real-world AI systems. It bridges classical NLP, deep learning, and modern LLM-based architectures, with emphasis on practical implementation and system-level thinking.

Each phase is organized into daily notebooks. Every day focuses on a specific concept, its intuition, implementation, and its connection to modern AI systems.

---

## Structure

- Phase-based progression
- Each phase contains multiple day-wise notebooks
- Each notebook includes:
  - Conceptual understanding
  - Practical implementation
  - Interview-focused insights
  - Mapping to GenAI / LLM systems
  - Mini-projects

---

## Phase 1 — Text Processing and Tokenization

Focus: Understanding how raw text is transformed into machine-understandable input and how modern LLMs interpret text.

Topics covered:
- Text preprocessing and normalization
- Handling punctuation, stopwords, and noise
- When not to preprocess in modern LLM pipelines
- Tokenization fundamentals (word, character, sentence)
- Subword tokenization (BPE, WordPiece, SentencePiece)
- Vocabulary construction and Out-of-Vocabulary problem
- Token distribution and Zipf’s Law (intuitive understanding)

Outcome:
Ability to understand and control how text is represented before entering any NLP or LLM system, including cost and performance implications in token-based models.

---

## Phase 2 — Classical NLP and Efficient Text Modeling

Focus: Building strong intuition for non-LLM approaches and understanding when simpler models outperform large language models.

Topics covered:
- Bag of Words representation and its limitations
- TF-IDF and its importance in search and retrieval
- Cosine similarity and document similarity systems
- Naive Bayes for text classification (intuitive understanding)
- Logistic Regression for text classification
- Support Vector Machines (conceptual understanding)
- Comparative understanding of classical models

Outcome:
Ability to design efficient, low-cost NLP pipelines and make informed decisions about when to use classical methods instead of LLMs.

---

## Phase 3 — Embeddings and Semantic Representation

Focus: Understanding how meaning is captured numerically and how modern systems use embeddings.

Topics covered:
- Word2Vec (CBOW and Skip-Gram)
- GloVe (co-occurrence-based representations)
- FastText and handling subword information
- Semantic similarity and vector arithmetic
- Sentence embeddings and modern embedding models
- Vector databases and similarity search (FAISS/ChromaDB)

Outcome:
Ability to build semantic search systems, understand embedding-based retrieval, and implement the core of RAG pipelines.

---

## Phase 4 — Transformers and Large Language Models

Focus: Deep understanding of transformer architecture and how modern LLMs function.

Topics covered:
- Attention mechanism and its motivation
- Self-attention (Query, Key, Value)
- Multi-head attention
- Positional encoding
- Transformer architecture (end-to-end flow)
- Encoder models (BERT)
- Decoder models (GPT)
- Encoder-decoder models (T5)
- Training objectives (masked language modeling, next token prediction)
- Fine-tuning techniques (LoRA, prompt tuning, adapters)

Outcome:
Ability to understand, explain, and work with transformer-based models, including reasoning about their behavior and limitations.

---

## Phase 5 — Modern GenAI Systems and AI Agents

Focus: Building production-level AI systems using LLMs, retrieval, and agent architectures.

Topics covered:
- Prompt engineering strategies
- Retrieval-Augmented Generation (RAG)
- Embedding-based retrieval pipelines
- AI agents (tool usage, memory, planning)
- Hybrid NLP systems (rule-based + ML + LLM)
- When not to use LLMs
- Evaluation metrics (F1, BLEU, ROUGE, perplexity — practical understanding)
- Model optimization (quantization, distillation)
- Prompting vs fine-tuning trade-offs

Outcome:
Ability to design and build scalable, efficient, and intelligent AI systems, including RAG pipelines and autonomous agents.
